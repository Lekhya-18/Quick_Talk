# Quick_Talk
A modern Android chat application that allows users to send and receive messages instantly using Android Studio.

- SmartChatApp is a modern Android chat application built using **Java, XML, and Firebase**.  
- It provides user authentication and lays the foundation for a real-time chat system.
---
## 📱 Features
- 🔐 User Authentication (Login & Signup)
- 🎨 Clean and modern UI design
- 🚀 Firebase Integration
- 📂 Structured project architecture
- 🔄 Navigation between screens
---
## 🛠️ Tech Stack
- **Language:** Java
- **UI:** XML (Android Layouts)
- **Backend:** Firebase
    - Firebase Authentication
    - Firebase Realtime Database
- **IDE:** Android Studio
---
## 📸 Screens
- Welcome Screen
- Login Screen
- Signup Screen
- Main Screen (After Login)
